RESULTS


We trained and tested multiple models using the same data set and compared classification accuracy, false positive rate, true positive rate, precision and recall. We observed that the non linear rbf Kernel SVM model predicted the outcome with the highest accuracy. The accuracy predicted was 86.7, with the smallest false positive rate.

Logistic regression had the highest true positive rate, but it had a higher classification error and false positive rate.

Neural Net was the second best performing model, subsequently we compared a regular neural net model with a deep neural net and observed that the accuracy of the deep neural net was less than regular net model.

K nearest neighbor model was the worst performing model with the least classification accuracy and the highest false positive rate. We modified the K classifier and observed that the difference between training and testing accuracy was continuously decreasing and did not give a clearer result. We tuned the model by selecting a K value in the middle (K10) but did but did not see any significant improvement.
